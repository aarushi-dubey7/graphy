# Graphy Frontend

The frontend renders a typed `LessonSpec` returned by the backend and keeps the graphical, numerical, explanation, and step views synchronized.

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run test`
- `npm run build`

## Environment

Set `VITE_API_BASE_URL` if the backend is not running on `http://127.0.0.1:8000`.
